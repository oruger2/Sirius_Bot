# ルール

1. [Bot-Hosting.net コントロールパネル](<https://control.bot-hosting.net/server/a752a477/>) ではファイルを更新しないでください。コンフリクトの原因となります。
2. AI の利用は問題ありませんが、Sign-offをAI Agentが行うことは明確に禁止します。法的に責任を負うことができるのは人間のみです。
3. 使わなくなったブランチは、**必ず** 削除してください。
4. 編集前には、**必ず** pull してください。[^1]
5. 編集には、可能であれば **Visual Studio Code**（Webプレビュー版でも可）を利用してください。
6. Visual Studio Code を利用している場合は、可能であれば 拡張機能 _`Error Lens`_ をインストールしてください。
7. PRの作成時にCoderabbitの自動レビューが走ります。必ずResolveしましょう。

[^1]: Visual Studio Code では、サイドバーのGitタブから操作できます
